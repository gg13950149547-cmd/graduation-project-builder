# Visual Style Samples

Store user-provided thesis style sample images here when they are available as actual files.

Usage rule:

- Save sample screenshots or cropped style references here by category when possible.
- Prefer template pages or formatting-rule documents first.
- If the template or rules document do not define a fixed style clearly, use the stored sample images here for visual review and matching.
- These stored sample images act as the fallback visual authority for TOC, formula, figure, table, title page, header, footer, page number, and body-text appearance.

Suggested subcategories:

- toc
- formulas
- figures
- tables
- title-pages
- headers-footers
- body-text
